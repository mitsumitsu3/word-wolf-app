# word-wolf-app
管理者からプレイヤーにワード配布する。

多分今回のケースではお金と実装の労力が一番抑えられる構成のはず🤔

- WebSocket、API → AppSync
- 管理画面の認証 → Cognito
- DB → DynamoDB
- フロント → ReactでSPAにまとめてS3でホスティング
